# Visual Identity Notes

This repository is a standalone toolbox rather than an article-support archive.
The visual identity should communicate technical reliability, research reuse,
and practical R programming.

## Direction

- Treat the repository as a product page for reusable functions.
- Keep the language honest: this is a collection of sourceable scripts, not a
  formal R package.
- Highlight the strongest reusable functions first, especially
  `soil_attr_balance()` and `balanced_raster_tiles()`.
- Avoid personal machine paths in the README.

## Palette

| Role | Hex | Use |
| --- | --- | --- |
| Console black | `#0c1117` | Banner background and technical tone. |
| Soft code green | `#7fb7a3` | Primary accent for function/toolbox identity. |
| Raster blue | `#365f7f` | Spatial processing and raster emphasis. |
| Model ochre | `#7a5b2e` | Modelling and prediction emphasis. |
| Snow text | `#f7fbfd` | Primary text over dark background. |
